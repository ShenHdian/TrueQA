# 真心话抽卡对战小程序 — P0+P1 交付总览

## 本次完成
按逐项确认的需求规格 v1，完成 **P0 地基 + P1 单人抽卡**，单机可玩。

## 架构
- **前端**：微信小程序原生（WXML/WXSS/JS），3 tab（抽卡/题库/我的）+ 对战分包占位。
- **后端**：零依赖 Node 服务（纯 `http`），JSON 文件持久化，首次启动自动种子化。
- **实时对战**：WebSocket 路由已留占位，P3 启用。

## 如何运行（详见 README.md）
1. `cd server && node src/index.js` 启动后端
2. 微信开发者工具导入 `mp/` 目录，填 AppID / 测试号
3. 本地设置勾选「不校验合法域名」
4. 编译，点「抽卡」tab 即可单机抽卡
