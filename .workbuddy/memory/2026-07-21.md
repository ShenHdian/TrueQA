## 风险与注意
- node:sqlite 需服务器 Node ≥ 22.5（本机 22.22 可用）。过低则回退 better-sqlite3。
- 本环境无遗留 db.json，迁移未自动触发；**生产服务器有 db.json 会触发迁移**（已验证路径）。
- 运行时 `trueqa.db` 被 .gitignore 忽略，不会入库；本地测试脏数据已 SQL 清理，现仅 30 系统题。
- 工具 safe-delete 在本机 Recycle Bin 故障时会 FAIL CLOSED 拒绝删文件——勿强行删运行时 db，用 SQL 清理即可。
